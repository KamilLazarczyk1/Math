Second-Order Linear ODEs — Undetermined Coefficients

Constant-coefficient equations $y'' + by' + cy = g(t)$: building the homogeneous solution from the characteristic equation, guessing a particular solution, and pinning down constants with initial conditions.

The method, from the ground up

What kind of equation is this?

Every problem on this sheet is a second-order linear ODE with constant coefficients:

$$ a\,y'' + b\,y' + c\,y = g(t), \qquad a,b,c \text{ constants},\ a\neq 0. $$

"Second-order" = the highest derivative is $y''$. "Linear" = $y, y', y''$ appear only to the first power, never multiplied together. "Constant coefficients" = the numbers $a,b,c$ don't depend on $t$. The right-hand side $g(t)$ is called the forcing term. When $g(t)=0$ we call it homogeneous; otherwise non-homogeneous.

The key structural fact

The general solution of the non-homogeneous equation is always

$$ y(t) = y_h(t) + y_p(t), $$

where $y_h$ (the homogeneous solution) is the full general solution of $a y''+by'+cy=0$, and $y_p$ (a particular solution) is any single function that satisfies the full equation with $g(t)$ on the right. Intuitively: $y_h$ carries the two free constants $c_1,c_2$ (a 2nd-order equation needs two), and $y_p$ takes care of the forcing.

Step 1 — the homogeneous solution via the characteristic equation

Try $y=e^{rt}$ in $a y''+by'+cy=0$. Since $y'=re^{rt}$ and $y''=r^2 e^{rt}$, dividing by $e^{rt}$ gives the characteristic equation

$$ a r^2 + b r + c = 0. $$

Its roots decide the shape of $y_h$:

Roots of $ar^2+br+c=0$Homogeneous solution $y_h$
Two distinct real roots $r_1\neq r_2$$c_1 e^{r_1 t} + c_2 e^{r_2 t}$
One repeated real root $r$$(c_1 + c_2 t)\,e^{rt}$
Complex pair $r=\alpha\pm\beta i$$e^{\alpha t}\big(c_1\cos\beta t + c_2\sin\beta t\big)$

The complex case comes from Euler's formula $e^{i\beta t}=\cos\beta t + i\sin\beta t$; the real and imaginary parts give the two real solutions.

Step 2 — a particular solution by "undetermined coefficients"

The idea: if $g(t)$ is a polynomial, an exponential, a sine/cosine, or a product of these, then $y_p$ has the same shape. We write down a guess with unknown coefficients, plug it in, and match. Here is the guess table:

Forcing term $g(t)$Trial guess $y_p$
$P_n(t)$ (polynomial of degree $n$)$A_n t^n + \dots + A_1 t + A_0$ (full polynomial of degree $n$)
$e^{at}$$A e^{at}$
$\cos(bt)$ or $\sin(bt)$$A\cos bt + B\sin bt$ (always include both!)
$e^{at}P_n(t)$$e^{at}(A_n t^n + \dots + A_0)$
$e^{at}\cos bt$ or $e^{at}\sin bt$$e^{at}(A\cos bt + B\sin bt)$

If $g(t)$ is a sum of such pieces, guess the sum of the corresponding trials (and solve piece by piece — this is the superposition principle).

Step 3 — the modification ("resonance") rule

The guess above fails if it is already a homogeneous solution (you'd just get $0=g$). The fix: multiply the trial by $t^s$, where $s$ is how many times the relevant root appears in the characteristic equation.

Concretely, look at the exponent that the forcing term "wants": for $e^{at}$ the value is $r=a$; for $\cos bt,\sin bt$ it is $r=\pm bi$; for $e^{at}\cos bt$ it is $r=a\pm bi$. Then

$$ s = \text{multiplicity of that }r\text{ as a root of } ar^2+br+c=0 \in\{0,1,2\}. $$

$s=0$ means no clash (use the plain guess), $s=1$ multiply by $t$, $s=2$ multiply by $t^2$.

Exercise 1 — Write an appropriate guess for $y_p$

Here we only set up the trial form (with the resonance rule applied); we are told not to solve for the coefficients. The recipe each time: find the characteristic roots, then read the guess off the table and check for a clash.

a)   $y'' + y = 4e^{3t} + 5t^2$

Roots. $r^2+1=0 \Rightarrow r=\pm i$. So $y_h = c_1\cos t + c_2\sin t$ (complex pair $\alpha=0,\beta=1$).

Piece $4e^{3t}$: wants $r=3$, which is not a root ($s=0$) $\Rightarrow$ guess $A e^{3t}$.

Piece $5t^2$: polynomial of degree $2$; it "wants" $r=0$, not a root ($s=0$) $\Rightarrow$ guess $Bt^2+Ct+D$.

Guess: $\quad y_p = A e^{3t} + Bt^2 + Ct + D.$

b)   $y'' - 5y' - 6y = 3e^{-2t} + 4\cos(3t)$

Roots. $r^2-5r-6=(r-6)(r+1)=0 \Rightarrow r=6,\,-1$.

Piece $3e^{-2t}$: wants $r=-2$, not a root $\Rightarrow A e^{-2t}$.

Piece $4\cos 3t$: wants $r=\pm 3i$, not roots $\Rightarrow B\cos 3t + C\sin 3t$.

Guess: $\quad y_p = A e^{-2t} + B\cos 3t + C\sin 3t.$

c)   $y'' - 2y' + 5y = 3t\,e^{t}$

Roots. $r^2-2r+5=0 \Rightarrow r=\dfrac{2\pm\sqrt{4-20}}{2}=1\pm 2i$.

Forcing $3t\,e^{t}$ is $e^{at}P_1(t)$ with $a=1$. It wants $r=1$, which is not a root (the roots are $1\pm 2i$, complex) $\Rightarrow s=0$. Use $e^{t}\times(\text{degree-1 polynomial})$.

Guess: $\quad y_p = (A t + B)\,e^{t}.$

d)   $y'' - 4y' - 5y = 3e^{2t}\sin(t)$

Roots. $r^2-4r-5=(r-5)(r+1)=0 \Rightarrow r=5,\,-1$.

Forcing $3e^{2t}\sin t$ is $e^{at}\sin bt$ with $a=2, b=1$. It wants $r=2\pm i$, not roots $\Rightarrow s=0$.

Guess: $\quad y_p = e^{2t}\big(A\cos t + B\sin t\big).$

Exercise 2 — Find the general solution

Now we carry out all three steps fully and solve for the coefficients. The general solutions below are written in the same order as the official answer key, so the constants line up exactly.

2a)   $y'' - y' - 12y = 10e^{5t}$

Homogeneous. $r^2-r-12=(r-4)(r+3)=0 \Rightarrow r=4,-3$, so $y_h = c_1 e^{-3t} + c_2 e^{4t}$.

Guess. $5$ is not a root, so $y_p = A e^{5t}$. Then $y_p' = 5Ae^{5t}$, $y_p'' = 25Ae^{5t}$.

Substitute & match.

$$ 25A - 5A - 12A = 10 \;\Rightarrow\; 8A = 10 \;\Rightarrow\; A = \tfrac{5}{4}. $$
Answer (matches key): $\quad y = c_1 e^{-3t} + c_2 e^{4t} + \dfrac{5}{4}e^{5t}.$

2b)   $y'' + y' - 2y = 4t^2 - 1$

Homogeneous. $r^2+r-2=(r+2)(r-1)=0 \Rightarrow r=-2,1$, so $y_h = c_1 e^{-2t} + c_2 e^{t}$.

Guess. RHS is a degree-2 polynomial, $r=0$ not a root, so $y_p = At^2+Bt+C$, with $y_p'=2At+B$, $y_p''=2A$.

Substitute.

$$ 2A + (2At+B) - 2(At^2+Bt+C) = -2A\,t^2 + (2A-2B)\,t + (2A+B-2C). $$

Match to $4t^2 + 0\cdot t - 1$:

$$ -2A = 4 \Rightarrow A=-2;\quad 2A-2B=0 \Rightarrow B=-2;\quad 2A+B-2C=-1 \Rightarrow -6-2C=-1 \Rightarrow C=-\tfrac{5}{2}. $$
Answer (matches key): $\quad y = c_1 e^{-2t} + c_2 e^{t} - 2t^2 - 2t - \dfrac{5}{2}.$

2c)   $y'' + 4y' + 3y = 2t + 4\cos t$

Homogeneous. $r^2+4r+3=(r+1)(r+3)=0 \Rightarrow r=-1,-3$, so $y_h = c_1 e^{-3t} + c_2 e^{-t}$.

Superposition. Solve for the two pieces of the RHS separately.

Piece $2t$: guess $y_{p1}=At+B$ ($y'=A$, $y''=0$). Substituting: $0 + 4A + 3(At+B) = 3A\,t + (4A+3B)$. Match $2t+0$:

$$ 3A=2 \Rightarrow A=\tfrac{2}{3};\qquad 4A+3B=0 \Rightarrow B=-\tfrac{8}{9}. $$

Piece $4\cos t$: guess $y_{p2}=C\cos t + D\sin t$. Then $y_{p2}'=-C\sin t+D\cos t$, $y_{p2}''=-C\cos t-D\sin t$. Substituting:

$$ (-C\cos t-D\sin t)+4(-C\sin t+D\cos t)+3(C\cos t+D\sin t) = (2C+4D)\cos t + (2D-4C)\sin t. $$

Match $4\cos t + 0\sin t$: from $2D-4C=0$ we get $D=2C$, then $2C+4(2C)=4 \Rightarrow 10C=4 \Rightarrow C=\tfrac{2}{5},\ D=\tfrac{4}{5}$.

Answer (matches key): $\quad y = c_1 e^{-3t} + c_2 e^{-t} + \dfrac{2}{3}t - \dfrac{8}{9} + \dfrac{2}{5}\cos t + \dfrac{4}{5}\sin t.$

2d)   $y'' + 4y' + 4y = t^2$

Homogeneous. $r^2+4r+4=(r+2)^2=0 \Rightarrow r=-2$ (repeated), so $y_h = (c_1 + c_2 t)e^{-2t}$.

Guess. Degree-2 polynomial, $r=0$ not a root, $y_p=At^2+Bt+C$, $y_p'=2At+B$, $y_p''=2A$.

Substitute.

$$ 2A + 4(2At+B) + 4(At^2+Bt+C) = 4A\,t^2 + (8A+4B)\,t + (2A+4B+4C). $$

Match $t^2+0t+0$: $4A=1\Rightarrow A=\tfrac14$; $8A+4B=0\Rightarrow B=-\tfrac12$; $2A+4B+4C=0\Rightarrow \tfrac12-2+4C=0\Rightarrow C=\tfrac38$.

Answer (matches key): $\quad y = (c_1+c_2 t)e^{-2t} + \dfrac{1}{4}t^2 - \dfrac{1}{2}t + \dfrac{3}{8}.$

2e)   $y'' + 4y = 20e^{t}\cos t$

Homogeneous. $r^2+4=0 \Rightarrow r=\pm 2i$, so $y_h = c_1\sin 2t + c_2\cos 2t$.

Guess. Forcing wants $r=1\pm i$ (not $\pm 2i$), so no resonance: $y_p = e^{t}(A\cos t + B\sin t)$.

Derivatives (product rule). With $y_p=e^t(A\cos t+B\sin t)$,

$$ y_p' = e^{t}\big[(A+B)\cos t + (B-A)\sin t\big], \qquad y_p'' = e^{t}\big[2B\cos t - 2A\sin t\big]. $$

Substitute into $y_p''+4y_p$:

$$ e^{t}(2B\cos t - 2A\sin t) + 4e^{t}(A\cos t + B\sin t) = e^{t}\big[(4A+2B)\cos t + (4B-2A)\sin t\big]. $$

Match $20e^{t}\cos t$ (so $20\cos t + 0\sin t$): $4B-2A=0\Rightarrow A=2B$, then $4(2B)+2B=20\Rightarrow 10B=20\Rightarrow B=2,\ A=4$.

Answer (matches key): $\quad y = c_1\sin 2t + c_2\cos 2t + 2e^{t}\sin t + 4e^{t}\cos t.$

2f)   $y'' + y' - y = 3$

Homogeneous. $r^2+r-1=0 \Rightarrow r=\dfrac{-1\pm\sqrt{5}}{2}$ (two distinct real roots), so

$$ y_h = c_1\,e^{\frac{-1-\sqrt5}{2}\,t} + c_2\,e^{\frac{-1+\sqrt5}{2}\,t}. $$

Guess. RHS is the constant $3$ (degree-0 polynomial); $r=0$ is not a root, so $y_p = A$. Then $y_p'=y_p''=0$ and the equation becomes $-A=3\Rightarrow A=-3$.

Answer (matches key): $\quad y = c_1\,e^{\frac{-1-\sqrt5}{2}\,t} + c_2\,e^{\frac{-1+\sqrt5}{2}\,t} - 3.$

Exercise 3 — Initial value problems

We take each general solution from Exercise 2 and impose $y(0)$ and $y'(0)$. This gives two linear equations for $c_1,c_2$. Recall $e^{0}=1$, $\cos 0=1$, $\sin 0=0$.

3a)   $y(0)=2,\ y'(0)=-1$  (equation 2a)

$y = c_1 e^{-3t} + c_2 e^{4t} + \tfrac54 e^{5t}$, so $y' = -3c_1 e^{-3t} + 4c_2 e^{4t} + \tfrac{25}{4}e^{5t}$.

$$ y(0)=c_1+c_2+\tfrac54 = 2 \;\Rightarrow\; c_1+c_2=\tfrac34, $$ $$ y'(0)=-3c_1+4c_2+\tfrac{25}{4}=-1 \;\Rightarrow\; -3c_1+4c_2=-\tfrac{29}{4}. $$

Solving: from the first $c_1=\tfrac34-c_2$; substitute to get $7c_2=-5$, so $c_2=-\tfrac57$ and $c_1=\tfrac{41}{28}$.

$c_1=\dfrac{41}{28},\ c_2=-\dfrac57$ — matches key.

3b)   $y(0)=1,\ y'(0)=1$  (equation 2b)

$y = c_1 e^{-2t} + c_2 e^{t} - 2t^2-2t-\tfrac52$, so $y' = -2c_1 e^{-2t} + c_2 e^{t} - 4t - 2$.

$$ y(0)=c_1+c_2-\tfrac52=1 \Rightarrow c_1+c_2=\tfrac72,\qquad y'(0)=-2c_1+c_2-2=1 \Rightarrow -2c_1+c_2=3. $$

Subtracting: $3c_1=\tfrac12\Rightarrow c_1=\tfrac16$, then $c_2=\tfrac72-\tfrac16=\tfrac{10}{3}$.

$c_1=\dfrac16,\ c_2=\dfrac{10}{3}$ — matches key.

3c)   $y(0)=2,\ y'(0)=0$  (equation 2c)

$y = c_1 e^{-3t} + c_2 e^{-t} + \tfrac23 t - \tfrac89 + \tfrac25\cos t + \tfrac45\sin t$, so $y' = -3c_1 e^{-3t} - c_2 e^{-t} + \tfrac23 - \tfrac25\sin t + \tfrac45\cos t$.

$$ y(0)=c_1+c_2-\tfrac89+\tfrac25 = 2 \;\Rightarrow\; c_1+c_2=\tfrac{112}{45}, $$ $$ y'(0)=-3c_1-c_2+\tfrac23+\tfrac45 = 0 \;\Rightarrow\; -3c_1-c_2=-\tfrac{22}{15}=-\tfrac{66}{45}. $$

Adding the two equations: $-2c_1=\tfrac{46}{45}\Rightarrow c_1=-\tfrac{23}{45}$, then $c_2=\tfrac{112}{45}+\tfrac{23}{45}=\tfrac{135}{45}=3$.

$c_1=-\dfrac{23}{45},\ c_2=3$ — matches key.

3d)   $y(0)=5,\ y'(0)=3$  (equation 2d)

$y = (c_1+c_2 t)e^{-2t} + \tfrac14 t^2 - \tfrac12 t + \tfrac38$. Differentiating the product, $y' = e^{-2t}\big(c_2 - 2c_1 - 2c_2 t\big) + \tfrac12 t - \tfrac12$.

$$ y(0)=c_1+\tfrac38=5 \Rightarrow c_1=\tfrac{37}{8}, \qquad y'(0)=c_2-2c_1-\tfrac12=3 \Rightarrow c_2 = \tfrac72 + 2c_1 = \tfrac72+\tfrac{37}{4}=\tfrac{51}{4}. $$
$c_1=\dfrac{37}{8},\ c_2=\dfrac{51}{4}$ — matches key.

3e)   $y(0)=0,\ y'(0)=-1$  (equation 2e)

$y = c_1\sin 2t + c_2\cos 2t + 4e^{t}\cos t + 2e^{t}\sin t$. Then $y' = 2c_1\cos 2t - 2c_2\sin 2t + 4e^{t}(\cos t-\sin t) + 2e^{t}(\sin t+\cos t)$.

$$ y(0)=c_2 + 4 = 0 \Rightarrow c_2=-4, \qquad y'(0)=2c_1 + 4 + 2 = -1 \Rightarrow 2c_1=-7 \Rightarrow c_1=-\tfrac72. $$
$c_1=-\dfrac72,\ c_2=-4$ — matches key.

3f)   $y(0)=-1,\ y'(0)=-1$  (equation 2f)

Write $r_1=\tfrac{-1-\sqrt5}{2},\ r_2=\tfrac{-1+\sqrt5}{2}$, so $y=c_1 e^{r_1 t}+c_2 e^{r_2 t}-3$ and $y'=c_1 r_1 e^{r_1 t}+c_2 r_2 e^{r_2 t}$. Note $r_1+r_2=-1$ and $r_1 r_2=-1$.

$$ y(0)=c_1+c_2-3=-1 \Rightarrow c_1+c_2=2, \qquad y'(0)=r_1 c_1 + r_2 c_2 = -1. $$

Try the symmetric guess $c_1=c_2=1$: then $c_1+c_2=2$ ✓ and $r_1 c_1 + r_2 c_2 = r_1+r_2 = -1$ ✓. Both equations hold, so this is the (unique) solution.

$c_1=1,\ c_2=1$ — matches key.
